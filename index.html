// 기술적 도전

HIGHLIGHTS

01
BT + FSM 하이브리드 AI
Switch-Case FSM에서 시작해 추상 클래스 기반으로 고도화했습니다. 이후 프레임 단위 상태 갱신으로 스택 오버플로우를 방지하고, BT에 가깝게 구조를 업그레이드 한 후 Unity GraphView API로 시각적 디버깅 툴을 직접 제작 중입니다.
→ 범용 하이브리드 상태 관리 시스템 구축 중
02
자체 충돌 파이프라인
OnTriggerEnter2D 기반의 터널링 현상을 CCD + RayCast로 해결했습니다 지금은 Unity Job System 기반 멀티스레드 Broad/Narrow Phase를 통한 충돌 파이프라인을 직접 설계 중입니다.
→ 탄막 다수 환경에서도 성능 병목 없는 시스템 목표
03
데이터 기반 증강 시스템
IAugment + MonoBehaviour 합성으로 생명주기 통합 관리하여 증강을 구현했습니다. 지금은 JSON + CSV 인덱스 포맷 + ScriptableObject이용으로 비개발자도 밸런스를 수정할 수 있는 환경을 구축했습니다.
→ 확장 가능한 설계를 지향
